7.8.3. Allocating DTC Objects to Control Module Descriptions

KWP DTC memory entries must be assigned per drag and drop to a KWP control module object in the control module tree.

UDS control module objects contain the allocation of DTC objects through their own import. If a DTC object is missing when importing control module descriptions, it is created during the control module import.
