### <a id="standard"></a>5.5. Default Texts

Default texts are centrally managed in ODIS Creator and can then be used by the author. In the default text view (see [Figure 74, “Default text view”](5-5-default-texts.md#figure.standard.navigator "Figure 74. Default text view")), there is the option to manage text groups and default texts. The tole of text administrator is necessary for this. The display of text groups and default texts is dependent on the brand and team assignment of the user. Default texts and text groups can only be created and edited by text administrators (see [Section 3.3, “Roles and Teams”](3-3-roles-and-teams.md "3.3. Roles and Teams")).
