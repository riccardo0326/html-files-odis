6.4.8. Monitoring Translations

ODIS Creator displays the translation packages organized by language in the translation navigator. They will be highlighted in color based on the following conditions:

ConditionDescription
No highlightingThe translation package has not been ordered. The deadline is not impending yet.
GreenThe translation package must be delivered soon. This gives you an overview of the expected translation packages.
YellowThe translation package must be delivered today. This gives you the option to ask the translation provider about time-sensitive translations.
RedThe translation package is overdue. The translation package was not delivered on time, and this allows you to take the appropriate actions.

Table 36. Translation Status


For a better overview, you can limit the translation packages listed in the translation navigator based on their initial deployment deadline through the definition of an individual filter.
