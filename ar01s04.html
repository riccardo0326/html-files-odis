4. ODIS Creator Business Processes

ODIS Creator is the editing system that allows diagnostic objects to be created and edited. These diagnostic objects are required for the off-board diagnosis in the ODIS Service processing system within "Guided Fault Finding" and "Guided Functions".

ProcessDescription
AdministrationProcess Administration (see Section 5, “ODIS Creator Administration Process”) includes all tasks relevant for defining the user and access rights, for general and location-specific configuration of languages, and for performing data imports that were initiated manually.
Order managementThe order management process (see Section 6, “ODIS Creator Order Management Process”) contains all tasks necessary for defining and managing orders for editing, and translating editorial objects. In general, creating and changing editorial objects as well as all translation activities are initiated and controlled by orders. Administrative processes are not controlled through order management.
EditingThe editing process (see Section 7, “ODIS Creator Editing Process”) contains all functional tasks that are necessary to edit and check editorial objects, which are required for performing off-board diagnosis on the ODIS Service diagnostic processing system. This process represents the central functional core of the ODIS Creator business processes.
ProvisionThe supply process (see Section 8, “ODIS Creator Preparation Process”) provides the released editorial objects (diagnostic packages, hotfixes) in the Mirror Server 2 depot for distribution to the customers.
