#### <a id="sgbaum.DTC-Verwendungsstellen"></a>7.7.7. DTC Usage Locations

A possible use in a suspicion rule or in a symptom descriptions is displayed at every DTC object in the control module tree.

Suspicion rules are visible through the associated diagnostic objects in the **used by** table. Symptom descriptions are visible through the function test objects; see the following image:
