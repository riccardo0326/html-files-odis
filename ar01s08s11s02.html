8.11.2. Collaboration export properties view

The collaboration export properties are listed here. Different properties will be displayed depending on which status the selected collaboration export has.

If a collaboration export is in the In processing status, the values are recalculated at periodic intervals. The new values will be displayed by reloading the object on the user interface.

The following properties will be presented in the display:

  • Export name

  • Export start time (format example: “18.09.2020, 13:03:06”)

  • Name of export creator

  • Export status

  • List of brands (separated by commas)

  • Number of objects already exported (only for In processing)

  • Number of objects still to be exported (only for In processing)

  • Estimated remaining time in minutes (only for In processing)

  • Number of exported objects (only for Completed)

  • Uncompressed size of the export in kilobytes (only for Completed)

  • Compressed size of the export in kilobytes (only for Completed)

  • Export end time (format example: „31.10.2012, 13:23:06“) (only for Completed and Error)

  • Check sum for the German export file (only for Completed)

  • Check sum for the English export file (only for Completed)
