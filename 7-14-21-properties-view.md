#### <a id="Funktionstesteditor.Zusaetzliche_Views.Eigenschaften_View"></a>7.14.21. Properties (View)

The attributes of the selected action element are displayed in this view. They match the data that is displayed in the tool tip. See [Figure 538, “Properties View”](7-14-21-properties-view.md#figure.Funktionstesteditor.Zusaetzliche_Views.Eigenschaften_View "Figure 538. Properties View").

<a id="figure.Funktionstesteditor.Zusaetzliche_Views.Eigenschaften_View"></a>

![Image: function test editor - additional views - properties view](images/fte_Eigenschaften_View.png)

**Figure 538. Properties View**

The values in the “Property” and “Value” columns can be selected individually or collectively and can be copied to the clipboard. Right-clicking opens a context menu containing two options:

- Copy values

  The right-hand column “Value” of the selected properties is copied. This action can also be performed with the key combination **Ctrl + C**.
- Copy properties and values

  Both the “Property” and “Value” columns of the selected properties are copied. This action can also be performed with the key combination **Ctrl + Shift + C**.

See also [Figure 539, “Copy Properties and Values View”](7-14-21-properties-view.md#figure.Funktionstesteditor.Zusaetzliche_Views.Eigenschaften_Kopieren_View "Figure 539. Copy Properties and Values View").
