5.4.8. Changing Comments for a Control Module Later

The comments for control modules that were already imported can be changed using "Change comments" in the context menu or the global menu. This only applies for control modules that have the status “Completed”, "Error”, or "Completed with errors”. The comments are changed in a separate modal dialog. (See Figure 72, “Comment Change Dialog”). Comments must be entered. Leaving the field empty is not permitted. Once the change is made successfully, the new comments will be updated and displayed in the editor. (see Figure 73, “Updating Comments”)
