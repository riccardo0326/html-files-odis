#### <a id="d4e14675"></a>7.13.3. Brand Variable Status Model

If a new brand variable is created, then it will have the initial status of "In processing" and is "active". Through the "release" menu item, you can convert one or more brand variables to the "released" status. Brand variables, that have reached the "released" status, can no longer receive another status.

As soon as a brand variable receives the "released" status, it can still be edited, however the comments can only be changed. All other attributes and settings are fixed. The status does not change if there is a revision in the "released" status.
