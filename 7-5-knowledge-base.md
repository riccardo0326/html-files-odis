### <a id="wissensbasis"></a>7.5. Knowledge base<a id="d4e11819"></a>

A knowledge base contains a diagnostic object tree, usually for a specific vehicle model or version. Every diagnostic object is assigned to exactly one knowledge base, which can however be used at various locations in the diagnostic object tree within a knowledge base. For initial and final tests on a vehicle, that belong to the knowledge base, a function test can be assigned to a knowledge base as a module start test and module end test.
