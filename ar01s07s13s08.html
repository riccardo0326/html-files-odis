7.13.8. Brand Variable Filters

The filter in the "brand variables" navigator functions the same as in other locations, such as the editor. Brand variables are filtered under the brand nodes.

The brand variable attributes that can be used for the filter can be found in Table 67, “Brand Variable Filter Attributes”.

System name=; LIKE; <>Free text
Variable type=; <>
