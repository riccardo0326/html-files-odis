## <a id="prozess.bereitstellung"></a>8. ODIS Creator Preparation Process

The **preparation** process prepares the released editorial objects (diagnostic packages (production, rollout and test), hotfixes, EU5 export packages) in the [Mirror Server 2](10-odis-creator-affiliated-systems.md#ria) depot for customer distribution. A detailed description is given below for packages in which distribution is managed differently. The settings in the process are brand-specific and are managed by the location administrators.

In addition to the **production baselines and production settings** views, the system also offers the **test baselines, test settings, rollout baselines and EU5 export** views. These views are described in the following.
