6.2.6. Switching the Context for Editorial Objects

While processing editorial objects, it may turn out that an editorial object is assigned to an incorrect order step. This may be the case if, for example, the editorial object is assigned to a scheduled order with a market introduction date in the distant future. Then it is possible to assign an editorial object to another order step using the drag and drop method.

To do this, select one or more editorial objects in the list of editorial objects for an order step and pull them to the desired order step in the order navigator. After confirming the confirmation prompt (Figure 127, “Confirmation Prompt when Switching the Context”), the order context of the selected editorial objects will be reset.

Figure 127. Confirmation Prompt when Switching the Context


Switching the context is not possible if the editorial object has one of the following statuses:

  • Being completed

  • In testing

  • In review
