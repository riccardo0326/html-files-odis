### <a id="datensprache"></a>9.3. Data Language

In the selection field under the tool bar ([Figure 644, “Data Language Selection Field”](9-3-data-language.md#figure.datensprache "Figure 644. Data Language Selection Field")), the user has the option to set the data language for the data being used. The fa-IR Farsi language is an exception. Farsi is always supported by OC. Due it its particular structure, Farsi is however filtered out of the data language selection.
