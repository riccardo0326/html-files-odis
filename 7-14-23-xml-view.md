#### <a id="Funktionstesteditor.Zusaetzliche_Views.XML_View"></a>7.14.23. XML View

The conversion from the graphic view is visible within the XML view. The XML view was implemented specially on request from Volkswagen for test purposes. See [Figure 541, “XML View”](7-14-23-xml-view.md#figure.Funktionstesteditor.Zusaetzliche_Views.XML_View "Figure 541. XML View").

In the view, only the text is displayed as a valid XML without colored markings or additional XML functions. The text can be marked and copied in order to add it in an XML editor and save it as an XML. To search in the function test, switch back to the editor or activate another function test view.
