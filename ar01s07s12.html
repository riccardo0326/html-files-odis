7.12. XML Templates

XML templates are treated like editorial objects. They will therefore obtain a version, and follow the standard release process of editorial objects. XML templates can only be imported and edited by you, if you have both internal roles of „author“ and „location administrator“ at the same time. The „author“ role is sufficient for using and viewing XML templates.

The import function is in the administration view in the menu structure. For the editing function of XML templates, there is an entry in the view selection bar on the left side when in the editing view (see Figure 324, “Entry in the View Selection Bar”). When this bar is used, the respective XML template navigator will open where all XML templates in the system will be displayed, for which you have the required authorization. If you select an XML template in the view, a read-only editor is opened (as with all other editorial objects), in which the metadata for the XML template is displayed. By double-clicking or using the context menu, you can open the editor for a selected XML template to edit it, depending on the authorization („author“ role is minimum requirement).
