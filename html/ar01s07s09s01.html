7.9.1. Importing Vehicle Projects

The vehicle projects themselves are loaded into ODIS Creator through a master data import („risk-released“ status) and are only available as read-only information.

It is possible to import generic function codes that work with vehicle projects and control module descriptions not yet released for the market. The vehicle projects imported via this mechanism are integrated in the tree for the remaining vehicle projects. Unreleased vehicle projects („in processing“ status) are indicated with the symbol (1 in Figure 291, “Vehicle Project Tree”).

The release of objects occurs through a regular import based on the applicable test criteria there. A manual release of vehicle projects by a user is explicitly not intended.
