6.4.9. Expert Functions for Translation Management

The purpose of expert functions for translation management is to be able to import translation packages repeatedly and to import translation packages that were not previously exported from the ODIS Creator system. The expert functions can only be started by the group administrator.

The expert mode is activated by pressing the "expert mode translation" icon in the administration. As soon as the button is pressed, the import of all of the packages in the Import_Expert folder will start.

The normal import of translation packages is only possible if export packages were previously created for the import packages. This is no longer required when expert mode is active.
