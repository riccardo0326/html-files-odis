7.7.5. Detailed View of Operations

By double clicking on an operation in the object editor for a control module object, for example, DiagnServi_Read DataByIdentCalibData, you will open a supplemental editor which has detailed information about this operation.

Here you will see all parameters and values of this operation in a structured overview. The structure contains the parameter classification as in or out parameters as well as the display of all values of the individual parameters. For example, you can see here the individual values of an enumeration of an out parameter.
