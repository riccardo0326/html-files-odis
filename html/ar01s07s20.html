7.20. Export and Import

The export and import serves to exchange data between two ODIS Creator environments. When exporting, the data will be stored in a file directory and when importing, the data will be imported from a file directory to a database and stored there.

Only function tests, general test and test procedures can be replaced during export and import. The file extension for import and export files conforms to ".oc23" and represents the ODIS Creator version.

As long as the correct view is in focus, the export and import functions are activated at the following locations in the user interface:

  • Global menu: File

  • Global button

  • Context menu of respective navigator
