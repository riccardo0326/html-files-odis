7.8.2. Importing DTC Objects

All DTC objects in the DTC memory tree are imported using the reference table, LT3 text table, or Cyclone text table. The reference table is created from group quality assurance information.

The DTC objects are read-only and therefore cannot be deleted. The DTC objects do not have a control module context here yet.

The system name semantics are portrayed as follows: „Decimal code [10]_P-Code[7]_Shortname“
