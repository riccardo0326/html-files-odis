8.11.4. Using an Export Template

Using export templates makes it possible to use export configurations from a collaboration export that was already completed in a new export. Pressing the “Select export template” button opens the dialog Figure 632, “Collaboration Export Template Selection Dialog” for selecting an export template.

Figure 632. Collaboration Export Template Selection Dialog


This dialog is laid out in a similar way to the collaboration export editor. Selecting a collaboration export loads the configuration of the export and fills in the dialog fields with this information. The dialog is only a read-only dialog and the configuration of the populated fields in the dialog cannot be changed. Clicking the OK button accepts the configurations in the collaboration export editor, where they can be edited. Any parameters selected in the collaboration editor will be overwritten.

The "Selected template” text field shows which template is currently selected.
