6.2.4. Usage Locations Dialog in Order Management

To access the usage locations dialog, select any editorial object within an order step and select the entry Display usage locations dialog in the context menu (see Figure 123, “Usage Locations Dialog Context Menu”). Alternatively you can select an object and then click on the button Edit and then Display usage locations in the menu bar. The usage locations dialog match the usage locations dialogs for the editorial objects in the "Edit" view. By default, only objects of the individual brand will be displayed without the navigation tree filter being activated, so that the editorial objects from every navigation tree can be displayed in the usage locations dialog (see Figure 124, “Usage Locations Dialog Order Management”). You can limit to the individual brand by checking the check box "only particular brand". To limit the results, filtering by navigation trees can be switched on through the respective combo box.

Figure 123. Usage Locations Dialog Context Menu


If more than one usage location exists for the selected editorial object, the usage locations dialog will open by double clicking on the selected editorial object.

When a listed usage location is selected, it will jump to the "Edit" view. The selected editorial object will be displayed at the corresponding location in the respective navigation tree. If the dialog is closed with the "Cancel" button, then the action will be canceled and you will remain in the "Order management" view.
