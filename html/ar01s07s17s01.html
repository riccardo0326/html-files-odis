7.17.1. Suspicion Rules

Suspicion rules can be determined in function tests and diagnostic objects. The suspicion rule type „EFA“ is to be used for the display, recording and editing of EFA codes in suspicion rules (see Figure 591, “Suspicion Rule Editor”).

Figure 591. Suspicion Rule Editor


After selecting the „EFA“ suspicion rule type, a button „…“ will be displayed when editing the „Value“ column (see Figure 592, “Starting EFA Coding Assist”), with which the desired EFA code can be determined by the EFA coding assist (see Figure 590, “EFA Coding Assist”).

Figure 592. Starting EFA Coding Assist
