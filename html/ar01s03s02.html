3.2. Program Structure

So that you can find your way easily in ODIS Creator Help, the program interface follows the familiar design of a Windows application. The ODIS Creator program name and your user name are given in the program title (1), the ODIS Creator symbol (2) to open the selection menu is located on the left side, and the symbols to minimize, maximize and close the program window (3) are on the right side. Under the window title, you will find the ODIS Creator menu bar (4) and under that is the tool bar (5), which contains the most important commands for quick access. The status bar is located at the lower edge of the program window (6), where information is displayed and a progress indicator is shown if you have started an action that takes longer to process.
