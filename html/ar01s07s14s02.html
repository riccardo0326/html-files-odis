7.14.2. Menus and Tool Bars

7.14.2.1. Main Menu Bar

The menu bar in the function test editor consists of the menu items for the editing environment and the menu items for the Editor. The functions can be called up using buttons in the menu line of the "Function test editor" screen. The following description of the menu items contains only the categories and actions that are relevant for the function test editor.

The menu bar contains the categories:

  • File

  • Edit

  • View

  • Insert

  • Extras

  • Help

Actions that cannot be performed are grayed out. See also Figure 335, “Menu Bar”.

Figure 335. Menu Bar
