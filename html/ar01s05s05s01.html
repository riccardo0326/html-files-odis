5.5.1. Manage Text Groups

The editor for creating a text group can be seen in Figure 75, “Text groups editor”. The user must give a title for the text group.

Figure 75. Text groups editor


Table 21, “Text Group Functions” describes the possible functions of the text group.

FunctionDescription
New text groupRegardless of the current group selection or a text in the navigator, a new text group can be created using the "text group" entry in the context menu. This function can also be accessed through the global menu "Insert > Text group".
Edit text groupYou can rework them by double clicking on an entry in the navigator. In addition to double clicking, this functionality can be accessed through the menu item "File > Edit".
