2.1.2. Help Overview

So that you can find your way easily in ODIS Creator Help, the interface follows the popular design of Windows Help. You will find the selection menu with the ODIS Creator symbol on the left side (1) and the symbols to minimize, maximize or close the window are on the right side (2). There is a bar under that where the search function can be found (3). In the left frame (4), you can open four different views through the lower register tab (5) (Table of Contents, Search results, Links to related topics, Bookmark). In the right frame, the selected help texts will be displayed (6).

2.1.2.1. Using the Eclipse Help Function

In the frame on the left side, the Content tab () contains the content folder. Click on a topic to display it in the right frame (6).

With the arrow keys and usage steps within help can be reversed or restored incrementally.

Search

To search for a specific topic, you can enter a search term in the Search field. You can use the search area button to limit the search to topic areas that you are interested in. The GO button starts the search. In the left frame, the Search results tab () will automatically open. Clicking on a search result opens the topic in the right frame.

Synchronize

If a topic was found and searched sufficiently, you can click on the button Update / Display current topic () or click on the button Display in content folder () to open the Content tab again in the left frame. The currently opened topic is displayed in the content folder structure tree.

Functionalities

If you want to display the documentation for functionalities, that are disabled in the application, select the button Display all topics (). If you select that all topics should be displayed in the content folder, then the headings of the documentation will also be displayed in disabled activities in the content folder and in the search results.

2.1.2.2. Navigation within an Open Section

To navigate in an open section, additional buttons are given in ODIS Creator Help. The buttons are in both the header and footer sections. Next (1) and Back (2) switch to the next or previous section. The button Back to top (3) switches to the beginning of a section and the button To start (4) opens the section overview for help.

Colored and underlined texts (5) are links to sections with additional information. Clicking on this type of text will take you directly to the section with further information about the linked text. Clicking on the upper left back arrow (6) switches back to the linked text passage.
