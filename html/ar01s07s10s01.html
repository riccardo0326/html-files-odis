7.10.1. Supplemental Document Categories

7.10.1.1. Text

Additional texts are available as XML files and are edited with the internal editor. Additional texts in XML format also have a link to the document type object.

When saving an supplementary document that contains text, an information dialog will appear if the document contains double spaces. The dialog informs the user with the message: "This document contains double spaces. Where would you like to replace them?". The affected double spaces are marked in the document and the user can select from the following choices:

  • Everywhere

    All double spaces in the entire document will be replaced.

  • In list elements

    Double spaces will only be replaced in list elements.

  • In free text

    Double spaces will only be replaced in free text (outside of lists).

  • Do not replace

    No changes will be made; the document will be saved without changes.

  • Cancel

    The saving process is canceled and the Editor remains open.

7.10.1.2. Illustration

Additional images are edited with a hotspot-capable editor and can contain document references through hotspots. Hotspots are internal objects, that originate a reference to other supplemental documents from a supplemental image through the cursor image position.

7.10.1.3. Media

Additional media are general media objects, such as audio and video documentation. Currently the picture formats PNG and SVG are supported.
