5.2. Managing Master Data

The master data is managed in the Master data navigator view in a tree structure (2).

Clicking on an entry opens a linked table or text field with contents in the editor (1). If you would like to edit the contents of a table, the editor must be opened in editor mode.

The master data properties (3) are displayed in the Properties window.
