7.10. Supplemental documents

A supplemental document is an editorial object that can be assigned to one or more diagnostic objects, a function test or other equipment characteristics. In addition to the common metadata for editorial objects, which can also contain the link to other supplemental documents or external references, it can contain an supplemental text supplemental document, an supplemental image or a general media object. For use in ODIS Service, supplemental documents can be filtered by their different categories and displayed separately. For the image that an supplemental document can contain, hotspots can always also be defined and every hotspot can refer to an external link or another supplemental document. Using the "Do not print test step and category" checkbox, the user can configure if test steps and categories should be printed out in the operating system. The default setting is “not activated”.
