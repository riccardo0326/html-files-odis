8.11.1. Collaboration export navigation tree

The collaboration export navigation tree shows an “Export settings” item for managing global exclusions lists as well as all existing collaboration exports in a tree structure. You have the option to display the properties of collaboration exports as well as properties of export settings (single mouse click).

The collaboration exports are listed in the tree in a hierarchy according to brand, and within the brands, according to the year created and the baseline (production/test baseline). The export settings entry is the highest node in the navigation tree and is displayed parallel to the brand nodes.

The status of collaboration exports is taken from the status model for production baselines. The status Created, In processing, Completed and Error is permitted for collaboration exports.

New collaboration exports can be created in the Export definition context menu. The editor opens in write mode by clicking on it, and then the required configuration entries can be made.

All collaboration exports can be deleted from the database in the Delete context menu. This will also deleted the export results. It is not possible to delete collaboration exports with the In processing status.
