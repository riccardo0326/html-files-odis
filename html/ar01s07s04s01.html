7.4.1. Create New Brand

You can create a new brand,by selecting an already existing brand and then clicking on New parallel node > Brand in the context menu or click on Parallel node > Brand in the Insert selection menu.

The system name for a brand is set through the defined brands in the administration. For this reason, you will select one from a list. You will define the market availabilitythrough the allocation of the markets defined in the administration. On one hand, you can set in which markets the brand is available and on the other hand, you will also indirectly define in which languages this object should be translated. The markets can be defined for all basic featuresof the equipment network (brand, vehicle model, model year, version and engine). The markets of the respective superordinate basic feature define the number of markets that can be assigned to all of the subordinate basic features. The market selection influences the languages in which the object is available at the end. The object is only provided on ODIS Service in the languages that are assigned to the selected markets.

The selection of an additional market is done with the symbol. With the symbol, a marked can be removed.
