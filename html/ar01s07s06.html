7.6. Function Library

The function library refers to all functional objects. It allows direct access to it and its list. The functional objects are arranged in the navigation tree based on their type. The types of functional objects may be separated in two categories; one group of the objects can only be created in the function library (measured value tables, test procedures, preset measurements), and the other group of objects can also be created in other navigation trees in editing (function test, general test, traversal test). Objects in the second group can be created there, where they are technically integrated. Further information can be found in the following sections. It does not matter in which navigation tree context a functional object is created, the functional library shows you the entirety of all objects of the corresponding type visible to you.
