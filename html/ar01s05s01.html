5.1. Administrator Tasks

RoleDescription
Group administratorThe role of group administrator is responsible for managing group data and importing group-wide reference data. Tasks include the management of languages, markets, brands, editing locations, organization units, and user roles as well as the import of master component lists, reference tables, VWG reference tables, LT3 text tables, Cyclone text tables, VFA reference tables, and measured values standard display.
Location administratorThe location administrator role is responsible for the location-specific user management and the configuration of the editing location. The tasks include the configuration of the editing location (preparation and import of control module descriptions) and the management of users and teams as well as the server configuration.
Text administratorThe role of text administrator is responsible for creating and editing default texts and text groups.
