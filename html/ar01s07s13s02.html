7.13.2. Editor for a Brand Variable

In the editor, the system name for the brand variables must be given. This consists of two parts. The first part is the KVPS code for the brand followed by an underscore, which is predefined by the system. The second part can be set freely by the user. The system name may only consist of the character set "_0123456789aAbBcCdDeEfFgGhHiIjJkKlLmMnNoOpPqQrRsStTuUvVwWxXyYzZ".

It can be set through the "Variable" or "Array" option fields, whether the brand variable is a field variable or not. When the "Array" option is selected, the input option for the array length also appears.

The "Brand" column is also given in the "Used by" table. The KVPS codes of the responsible brands for the respective objects are given in this column.

Like at other locations in ODIS Creator, it is also possible to jump from the "Used by" table to the objects. For this, you will select the desired object in the table and it opens the corresponding navigator with the object and the object editor in read mode. It is necessary that you have the necessary rights to the target object for this.
