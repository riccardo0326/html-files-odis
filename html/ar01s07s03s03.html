7.3.3. ID Search

When searching for IDs, it will differentiate between version-specific and version-independent IDs. The IDs contained in the deployments are version-independent. To determine the editorial object assigned to these IDs, version-independent must be selected as the ID type for the search. The translation packages contain version-specific IDs. To determine the editorial object assigned to these IDs, version-specific must be selected as the ID type for the search. There is also the option to search for more than one ID at the same time. To do this, simply enter the IDs consecutively separated by a space or a comma.

You can select which type of ID should be searched for the specified ID content. The Figure 230, “Search for ID” shows the dialog for the search for IDs.
