7.13.7. Usage Locations Dialog

There are two options for accessing the usage locations dialog for brand variables. As at other locations in ODIS Creator, one option is to open the usage location dialog through the context menu in the "used by" table.

The second option is that you select a brand variable node and select "usage locations" in the context menu. The dialog to be opened is different from the normal usage locations dialog. The dialog is not modal, meaning it can be moved to the side to make other work in the ODIS Creator possible. If another brand variable is selected while the dialog is open, then the contents in the dialog are updated. Another difference is that this dialog does not have the "only particular brand" checkbox, but rather has a selection field (see Figure 333, “Brand Variable Usage Locations Dialog”).
