8.7. Logs for Baselines

Log files for a baseline can be viewed and downloaded in the baseline editor. All log files can be saved as an archive using the download button. The “Log” button used to display loads will load them in the editor. The individual log files with various information can then be selected for display using a drop-down box below the button. You can see an example of the status file for a production baseline in Figure 611, “Log Files View”.
