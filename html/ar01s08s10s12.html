8.10.12. Properties

If a test is selected in the navigator, then the respective properties are displayed in the properties window. The following properties are displayed with the corresponding values (see Figure 623, “Properties Window Test”):

  • Initiator: when tests are manually started, the first and last name of the location administrator (or the login code, if neither the first nor last name is set) that started the test. "System" is entered as the value for automatically started tests.

  • Test created: time stamp of the test object initialization.

  • Test started: time stamp when the test was actually started.

  • Test ended: time stamp when the test ended with or without errors.

  • Current status: the current test status. See also Section 8.10.5, “Test Status”
