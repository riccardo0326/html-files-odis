8.8. Changing Comments for a Baseline Later

See Section 5.4.8, “Changing Comments for a Control Module Later”.

Comments can be changed regardless of the baseline type, meaning they can be changed later for complete production baseline, incremental production baseline, and test baseline deployments. To do this, the baseline must be in a final status (released, error, canceled, created externally, in processing externally, completed externally, or empty).
