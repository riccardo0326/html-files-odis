5.2.8. Server Configuration

Under the Server configuration entry in the master data navigator (2), the location administrator has the option to display or edit the configuration files on the ODIS Creator Server using various editors. The files can also be locally imported or exported.

Please note the information and comments in the corresponding configuration files.
