3.2.2. Edited Order (Order Context)

It is displayed directly under the tool bar which order you currently have in editing (Figure 25, “Order Context Bar”).. Since deadlines are attached to the orders that could relate to specific content, it is important that you always know which order you are editing diagnostic data for.

Figure 25. Order Context Bar


If no order is displayed, a query is given by ODIS Creator as soon as you would like to create or change diagnostic information in ODIS Creator. You can set or change the edited order in each view by clicking on the icon and selecting the suitable order step within an order context.

The dialog to select a procedure context shows all of the order headers and order steps currently visible for you. For a better overview, you can access a filter to filter the entries. The filter is opened through a button in the context dialog. For this, the recognized filter from the navigation tree in the "orders" view is opened (see Figure 26, “Filter Button in the Context Dialog”). You can find more information about using the filter under Section 3.2.3.1, “Defining the filter”.

Figure 26. Filter Button in the Context Dialog


If you cannot find an applicable order, you always have the option to define an order yourself and to set it as the order context.
