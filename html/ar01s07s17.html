7.17. Standard Fault Description Language (EFA)

Error symptoms in vehicles and existing constraints in connection to that were previously summarized in the Volkswagen group by hierarchically structured codings from the DISS system. Due to the growing complexity of the error symptoms and increasing demand for the analysis of such errors, a new system called EFA (Standard Fault Description Language) was introduced, which can depict error symptoms together with their dependencies in a complex structure. All group systems which currently work with the DISS coding are converted to using the EFA coding. ODIS Creator is also affected by this conversion.

In the EFA system (Standard Fault Description Language), error symptoms are depicted in a complex structure together with their dependencies for the error analysis.

Figure 590. EFA Coding Assist


The EFA coding assist can be accessed at multiple locations in ODIS Creator. It appears as shown in Figure 590, “EFA Coding Assist”. This generally affects the editors for:

  • Suspicion Rules

  • External References
