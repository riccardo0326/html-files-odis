7.1.7. Editorial Object History

A version history with the version number, date, version comments and the baseline version, with which this version was first prepared, is listed for all editorial objects in the Editorial object history view. By double clicking on a version, it will be displayed in the editor window. Since an object can also be used in multiple brands, the brand name is prefixed as a KVPS number to the baseline version. There is no transformation of data, thus objects in this view might have no assigned baseline, with which they were provided. However, this does not necessarily mean that they were never provided.

For each version, the context (in the form of an order header/order step) is displayed in which the version is managed. The field is empty for versions that are not assigned to an order step (such as editorial objects from the migration).
