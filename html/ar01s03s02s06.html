3.2.6. Tables and Lists

In some tables,the order of the table rows can be changed (Figure 37, “Table Sorting”). To do this, click on the column header of the column containing the entries which the table should be resorted by. Based on the entries in the selected column, the table lines are displayed in ascending, descending or unsorted alphanumeric order.

Figure 37. Table Sorting


In some tables or lists you can create new lines, in which you click on the plus symbol on the right side above a table or list. If you would like to change an already existing entry, click directly in the respective line. To delete a line, mark it and then click on the minus symbol.

In the editing process, tables serve only to display contents. The cells cannot be edited directly. New data sets are created using the button . Then you have to option to create a link to the Internet, in the context menu for example. A supplemental editor is loaded, in which the contents are entered and then applied.
