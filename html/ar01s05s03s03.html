5.3.3. Assignment of Meta Roles

The assignment between roles and meta roles occurs within the role management interface, in which a selection menu with all meta roles is displayed in the dialog for the properties of a role, from which then the suitable meta roles can be selected. Since this is part of the user, team and roll administration, nothing more should be done at this point.
