7.12.3. Editor for XML Templates

If you select an XML template in the navigator, then the XML template is displayed in a new editor. The structure and functionality of the editor is designed like the editor for supplemental documents. All XML template properties are displayed in the editor and can be edited here. The corresponding editor with all properties is portrayed in Figure 328, “Editor for XML Templates”. You can set what the XML template should be called in the system name attribute. You can also use this to set which file name the XML template is provided under and the system name plus file extension „.xml“. The rest of the attributes behave the same as the standard attributes, which apply to all editorial objects. The actual content file can be displayed through the context menu or through the extra menu. If the editor is opened in read-mode, only the Display context menu item will be given. This menu item shows the content of the XML file in the editor that was specified in the settings.

The definition of the external editor is located under Help >> settings.

If the editor opens in write mode, the context menu item „Edit“ appears. The specified editor is opened for this and the XML content is displayed. You can revise the content, respectively. After finishing the editor, the content is saved in ODIS Creator after the save process was initiated by you. There is a schema for XML templates. Therefore, possible support functions that are provided for creating XML files in the defined editor cannot really be used, because it functions simply as a text editor.
