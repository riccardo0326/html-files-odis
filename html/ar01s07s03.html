7.3. Search

You can search for objects, content and translations in current supply in ODIS Creator. It will vary between the object search, full text search and ID search. To start the search, press the button combination Ctrl+F or select Edit >> Search in the menu bar or in the selection menu.

The search will be performed asynchronously in the background. You can perform other actions in the user interface while the search is being performed, however you cannot start another search. When the search has ended, you will be notified with a message and the search results can be displayed. If you are directly in the editing view, then the search results view will also automatically open immediately. If you are in another view, then the search results view will open as soon as you switch back to the editing view.

The search is divided into the following areas:

  • Section 7.3.1, “Object Search”

  • Section 7.3.2, “Full Text Search”

  • Section 7.3.3, “ID Search”

Note: Editorial objects from another brand can only be found when they are in the "risk-released" status.
