5.2.4. Creating a Brand

A new brand can be created in Administration > Group data > Brand. A description, an MS/2 label, a KVPS name, a GoCAT name and a headquarters location are required for this. This headquarters location is created under editing location. The brand is required to be able to create an editing location.

Due to this dependency between an editing location and brand, both of these objects must be created at the same time. Open both editors in write mode to do this. Create a new brand without the editing location and create an editing location without a brand and save both editors one time. You will receive an error message that no editing location or brand was specified. Now you can select the editing location in the brand editor and the brand in the editing location editor and save both.
