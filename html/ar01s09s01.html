9.1. Settings

Under Settings, under the Help menu item, the user has the option to set several configuration parameters.

Figure 637. Settings


SettingsDescription
Individual notification settingsHere you have the option to manage the period until the next notification reminder (Section 6.1.3.5, “Notification Reminder”).
Team notification settingsHere you have the option in an analog display to manage the period until the next notification reminder for the team, in which you are a member.
Filter settingsHere you have the option to set the parameters for saved filters (Section 3.2.3.4, “Saved Filter Administration”).
LoggingHere you have the option to set the granularity for the writing of logging information (Section 7.15.11.6, “Logging Mechanism Configuration”).
ODIS DebugHere you have the option to configure the interface updates for ODIS Debug (Section 7.15.5.3, “Interface Updates Configuration”).
Function Test EditorHere you have the option to set the parameters for the function test editor (Section 7.14.9.12, “Layout”).
Layout settingsHere you have the option to select the layout of the editors. This affects all object editors of the edition as well as the editors of orders, therefore the order header and order step (Section 9.1.2, “Layout Settings”).
Primary filter settingsHere you have the option to set for every editing navigator, whether only the navigator primary objects are displayed when opening or if all other referenced editorial objects should also be displayed. (Section 7.1.5.2, “Filter for Primary Objects”).
Location codeHere you have the option to set for different editor displays, whether the language code should be displayed after the system name or not (see Section 7.1.9, “Location Assignment of Editorial Objects”). In Figure 638, “Location Code Settings” you see the settings for the language code in the settings.
Text ID favorite settingsThere is the option here to manage auto completion favorites (see Section 5.6.3.5.2, “Favorites”).
XML template editorHere you have the option to set the path for the editor for editing XML templates (see Section 7.12.3, “Editor for XML Templates”). The default installation path for the Notepad++ editor is entered as the default path.
