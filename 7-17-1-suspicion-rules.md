#### <a id="efa.verdachtsregeln"></a>7.17.1. Suspicion Rules

Suspicion rules can be determined in function tests and diagnostic objects. The suspicion rule type „EFA“ is to be used for the display, recording and editing of EFA codes in suspicion rules (see [Figure 591, “Suspicion Rule Editor”](7-17-1-suspicion-rules.md#figure.efa.verdachtsregeln "Figure 591. Suspicion Rule Editor")).

<a id="figure.efa.verdachtsregeln"></a>

![Image: suspicion rule editor](images/verdachtsregel-editor.png)

**Figure 591. Suspicion Rule Editor**

After selecting the „EFA“ suspicion rule type, a button „…“ will be displayed when editing the „Value“ column (see [Figure 592, “Starting EFA Coding Assist”](7-17-1-suspicion-rules.md#figure.efa.start "Figure 592. Starting EFA Coding Assist")), with which the desired EFA code can be determined by the EFA coding assist (see [Figure 590, “EFA Coding Assist”](7-17-standard-fault-description-language-efa.md#figure.efa "Figure 590. EFA Coding Assist")).

<a id="figure.efa.start"></a>

![Image: starting EFA coding assist](images/wert-button.png)

**Figure 592. Starting EFA Coding Assist**
