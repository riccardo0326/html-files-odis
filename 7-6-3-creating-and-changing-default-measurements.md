#### <a id="voreingestelltemessung"></a>7.6.3. Creating and Changing Default Measurements<a id="d4e13426"></a>

A default measurement represents a preset measuring curve. The measuring curves are attached to the object as \*.sk2 files. The 'measuring equipment' attribute specifies if the comparison value may be used in the 'open' measuring equipment (separate measurement and default value entry). A default measurement is used within the test sequence of a function code object.

Default measurements can be created in the function library as subnodes of the 'default measurements' group node or as a parallel node to already default measurements. Simply select the corresponding object and select **New subnode >> Default measurement** or **New parallel node >> Default measurement** in the context menu.
