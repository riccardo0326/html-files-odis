### <a id="programmaufbau"></a>3.2. Program Structure

So that you can find your way easily in ODIS Creator Help, the program interface follows the familiar design of a Windows application. The **ODIS Creator** program name and your user name are given in the program title ([1](3-2-program-structure.md#figure.hauptfenster "Figure 23. ODIS Creator Main Window")), the ODIS Creator symbol ([2](3-2-program-structure.md#figure.hauptfenster "Figure 23. ODIS Creator Main Window")) to open the [selection menu](2-2-general-definitions-and-glossaries.md#auswahlmenu) is located on the left side, and the symbols to minimize, maximize and close the program window ([3](3-2-program-structure.md#figure.hauptfenster "Figure 23. ODIS Creator Main Window")) are on the right side. Under the window title, you will find the ODIS Creator [menu bar](2-2-general-definitions-and-glossaries.md#menuleiste) ([4](3-2-program-structure.md#figure.hauptfenster "Figure 23. ODIS Creator Main Window")) and under that is the [tool bar](2-2-general-definitions-and-glossaries.md#symbolleiste) ([5](3-2-program-structure.md#figure.hauptfenster "Figure 23. ODIS Creator Main Window")), which contains the most important commands for quick access. The status bar is located at the lower edge of the program window ([6](3-2-program-structure.md#figure.hauptfenster "Figure 23. ODIS Creator Main Window")), where information is displayed and a progress indicator is shown if you have started an action that takes longer to process.
