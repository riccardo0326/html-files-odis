#### <a id="redaktion.search.result"></a>7.3.4. Search Results

The editorial objects that meet the entered search criteria are displayed in the results view (see [Figure 231, “Search Results”](7-3-4-search-results.md#figure.suchergebnis "Figure 231. Search Results")) with the following columns:

- System name
- Status
- Type designation and object type icon
- Version
- Brand

The search result only includes the most current version of every found editorial object that can be viewed (exception: search for version-specific IDs).

<a id="figure.suchergebnis"></a>

![Image: search results](images/Suchergebnis.png)

**Figure 231. Search Results**

The text to be searched will not be found with the object search and full-text search:

- if it is only available in an old version, which has at least a more current version with a confirmed completed status or higher in existence,
- if it is only in the most recent version that has the in processing status, and the user is not a member of the responsible team,
- if the object is in the recycle bin.

When an editorial object is selected in the results view, a view with the full editorial object metadata opens the same as when selecting in the navigator tree. Double clicking on an entry will open the respective object editor in the "in processing" mode. The function "Mark in tree" also continues to be available in the results view, in order to mark the selected editorial object in the navigator tree.

##### <a id="redaktion.search.result.export"></a>7.3.4.1. Excel Export

The current search results can be exported to an Excel file with a button in the results view or with the "Excel export" entry in the menu. In addition to the object properties viewed in the search results, the respective version-independent ID of the found objects are entered into the Excel file.

After pressing the menu item, a file selection dialog appears, in which you are prompted to select the target folder as well as enter the file name. After confirming the selection dialog with "OK", an Excel file with the selected name is created and the complete search results are entered there.
