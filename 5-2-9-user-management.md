#### <a id="benutzerverwaltung"></a>5.2.9. User Management

The user is the central object. Users are grouped in teams. Every team is linked to a location and a role (see [Section 3.3, “Roles and Teams”](3-3-roles-and-teams.md "3.3. Roles and Teams")). The allocation of function rights is determined by the roles.

The function rights regulate which functions a user may perform.

Executing functions on partial data quantity can be limited by data-oriented rights (object rights). The allocation of object rights is based on the team.

##### <a id="zuordnung.objektrechte"></a>5.2.9.1. Allocation of Object Rights to the Teams

Special sub-objects for the object roles and team are formed for the system-defined roles, that result from functional requirements.
