## <a id="prozess.administration"></a>5. ODIS Creator Administration Process

The **administration** process includes all tasks relevant for defining the user and access rights, for general and location-specific configuration of languages, and for performing data imports that were initiated manually. The administration process is handled by the group administrator and location administrator roles.

The use cases can be separated into four groups:

- Group-wide reference data management

- Location-specific user management and configuration

- Import of group data

- Server Configuration
