#### <a id="prozess.bereitstellung.pruefung.vorlage"></a>8.10.11. Manual Test as Template

You can use a manual test already performed as a template for a new manual test. Select the performed test in the tree for this. Select "Use as template" through the context menu. A new editor will open in write mode and is populated with the values from the template.
