#### <a id="aufgaben.uebersetzungskoordinatoren"></a>6.1.2. Tasks for Translation Coordinators

The tasks for translation coordinators consist of translation candidates<a id="d4e3726"></a>in the various target languages and the translation packages<a id="d4e3728"></a>. ODIS Creator can autonomously determine the translation candidates, therefore all editorial objects and the texts from the administration.

It is determined in the administration, which languages are required for which markets. It is defined which markets they are relevant for the elements in the equipment network. By allocating the editorial objects in the equipment network, ODIS Creator can then determine which languages the information should be in. The translation candidates are derived from this, therefore all information that is determined for a specific market and therefore for one or more languages and not yet translated into the respective version.

So that every individual released text does not have to be separately commissioned for translation, ODIS Creator can automatically create a translation package. The criteria **"maximum retention time"**<a id="d4e3733"></a>and **"minimum volume"**<a id="d4e3736"></a>are considered for this. A translation package will then be automatically created, if the minimum volume per brand has been reached for a language and also for editorial objects or if at least one translation candidate has reached the maximum storage period.

If the translation candidate is an editorial object, that was created or changed within a scheduled order with a market introduction deadline, then a translation package can also be automatically created if neither the maximum storage period nor the minimum language volume has been reached. It should be determined if there is enough time until the market introduction for the translation. ODIS Creator can however not guarantee that the translation can be available and made public by the market introduction, since the translation may take longer than planned or the release is so close to the market introduction that there is no longer enough remaining time. ODIS Creator marks late translation packages.

Checking the causes for a delay is one of the translator coordinator's tasks. The second task is to pack required translation candidates immediately into a translation package and commission them without waiting for the automated package creation. This will make it possible to provide urgent information as quickly as possible to the factories.
