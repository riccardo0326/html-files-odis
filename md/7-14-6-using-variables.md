#### <a id="Funktionstesteditor.Debugger.FunktionstestVariablen_verwenden"></a>7.14.6. Using Variables

The **Variable** button is used to add variables or arrays from declare commands into expressions or message texts. The selection is sorted alphabetically based on the interface language. Local variables are listed separately from global variables. The following image shows the variable sorting in the selection list.

<a id="Funktionstesteditor.Debugger.FunktionstestVariablen_VariablenButton"></a>

![Image: variables button](images/fte_FunktionstestVariablen_VariablenButton.png)

**Figure 368. Variable Button**

The **Keyword** button is used to add keywords from define commands into expressions or message texts. The keywords are sorted alphabetically in the keyword list based on the selected interface language. Local keywords are listed separately from global keywords. The following image shows the keyword sorting in the selection list.

<a id="Funktionstesteditor.Debugger.FunktionstestVariablen_KennwortButton"></a>

![Image: variables button](images/fte_FunktionstestVariablen_KennwortButton.png)

**Figure 369. Keyword Button**

The definition of variables is found in [Section 7.14.11.1, “Action Element - Declare Variable”](7-14-11-category-definition.md#Funktionstesteditor.Arbeitsflaeche.Aktionselemente_Kategorie_Definition.Deklariere_Variablen "7.14.11.1. Action Element - Declare Variable").

The declaration of keywords is found in [Section 7.14.11.4, “Action Element - Define Keyword”](7-14-11-category-definition.md#Funktionstesteditor.Arbeitsflaeche.Aktionselemente_Kategorie_Definition.Definiere_Kennwort "7.14.11.4. Action Element - Define Keyword").
