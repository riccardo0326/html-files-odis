#### <a id="javabox.navigator"></a>5.8.1. Java Box Test Navigation Tree

The navigation tree for the Java box test shows an entry for each function code object to be tested that contains at least one unverified “Java box” action element.

The test requests are listed in the navigator with the name: timestamp\_system name, sorted according to creation time (see [Figure 106, “Java Box Test Display”](5-8-java-box-test.md#figure.javaboxcheck.overview "Figure 106. Java Box Test Display")).

Each time OC starts or the “Java box test” view is refreshed, a message informs the tester that there are test requests to be tested. See [Figure 107, “Test Requests to be Tested”](5-8-1-java-box-test-navigation-tree.md#figure.javabox.Pruefungen "Figure 107. Test Requests to be Tested").
