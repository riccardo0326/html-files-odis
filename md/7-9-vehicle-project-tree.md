### <a id="fahrzeugprojekte"></a>7.9. Vehicle Project Tree

The vehicle project tree contains all available vehicle projects in the group. All vehicle projects are sorted by brand in this view. You will find the vehicle project tree in the editing view under the DTC memory tree ([Figure 291, “Vehicle Project Tree”](7-9-vehicle-project-tree.md#figure.fahrzeugprojekt "Figure 291. Vehicle Project Tree")). The vehicle projects are structured under the group brands in the navigator.
