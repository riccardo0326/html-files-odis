#### <a id="prozess.bereitstellung.pruefung.status"></a>8.10.5. Test Status

An icon is displayed in front of the test name, that corresponds to the current test status. There are the following statuses:

- ![](images/baseline_created.png) Created

  If a new test is created, then the object first receives the status "Created".
- ![](images/baseline_in_progress.png) Running

  As soon as the corresponding Documentum job begins with the test, then the status switches to „Running“.
- ![](images/forbidden.png) Canceled

  If the test is canceled by the location administrator, then the test object receives the final status „Canceled“.
- ![](images/baseline_finished.png) Completed

  As soon as the test was successfully performed, the object receives the final status „Completed“.
- ![](images/baseline_failed.png) Error

  If the test is canceled due to an error, then the test object receives the final status „Canceled“.
