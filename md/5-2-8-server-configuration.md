#### <a id="administration.server.configuration"></a>5.2.8. Server Configuration

Under the **Server configuration** entry in the master data navigator ([2](5-2-managing-master-data.md#figure.stammdatennavigator "Figure 47. Master Data Navigator")), the location administrator has the option to display or edit the configuration files on the ODIS Creator Server using various [editors](3-2-5-editor.md "3.2.5. Editor"). The files can also be locally imported or exported.

Please note the information and comments in the corresponding configuration files.
