#### <a id="ausstattungsnetz.motorkennbuchstabe"></a>7.4.8. Determining the Engine Code from the Engine Control Module

It is possible to determine the engine code in the equipment network from the engine control module.<a id="d4e11691"></a>

The engine code can be selected with the following editorial object types:

- Vehicle type
- Version
- Motor

To determine the engine code and transfer it into the vehicle features dialog, the “Engine code via control module” checkbox must be selected. (see [Figure 242, “Engine Code via Vehicle Type Control Module”](7-4-8-determining-the-engine-code-from-the-engine-control-module.md#figure.redaktion.ausstattungsnetz.motorkennbuchstabe "Figure 242. Engine Code via Vehicle Type Control Module"))<a id="d4e11704"></a>

If the checkbox is not selected, then the engine control module will not be queried.
