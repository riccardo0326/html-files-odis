#### <a id="baseline.state.empty"></a>8.6.5. An incremental baseline is not required

This status shows that this baseline does not have to be rebuilt, since there have been no changes to the database since the last baseline.

The icon for this status shows ![](images/baseline_empty.png) a yellow exclamation point.
