### <a id="steuergeraetebaum"></a>7.7. Control Module Tree

The control module tree refers to all objects that describe the properties of control modules. It allows the direct access to these objects as well as their listing and hierarchical access through the root objects of the control module tree. The control module tree is automatically created and updated by the import.
