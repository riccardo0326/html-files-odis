#### <a id="standortspezifischer.export"></a>7.19.3. Location-Specific Export

The collection of use relationships to be exported occurs based on location. For the allocation of use placements to a location, the following method applies: every editorial object has a responsible team and every team belongs to a location. For every P-differ export, the marked knowledge base objects are used as root objects for the export files that were created by a team at this location.
