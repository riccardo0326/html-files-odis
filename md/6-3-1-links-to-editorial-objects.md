#### <a id="auftragsverwaltung.textbeauftragung.verlinkungen"></a>6.3.1. Links to Editorial Objects

There is a table with linked editorial objects in the “Text request” editor. (see [Figure 143, “Text Request Editor”](6-3-text-request.md#figure.textbeauftragungseditorautor "Figure 143. Text Request Editor"))

If you have the necessary user rights to view the object, then you can double click on an editorial object to switch to the corresponding content in the editing view. Similar to the existing link functions in other modules, the link function cannot be executed if you are not authorized to view and/or edit the editorial object.
