#### <a id="markenvariable.deaktivieren"></a>7.13.5. Deactivating/Activating Brand Variables

A brand variable can be deactivated and activated. To do this, select "Deactivate brand variable" or "Activate brand variable" in the brand variable context menu. An inactive brand variable is no longer displayed for selection of usable brand variables. Used, deactivated brand variables cause an error when saving function tests. Brand variables can be deactivated and reactivated at any time.
