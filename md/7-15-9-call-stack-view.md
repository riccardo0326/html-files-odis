#### <a id="Funktionstesteditor.Debugger.Callstack_View"></a>7.15.9. Call Stack View

The call stack view (see [Figure 581, “Call Stack View”](7-15-9-call-stack-view.md#figure.Funktionstesteditor.Debugger.Callstack_View "Figure 581. Call Stack View")) shows the current call stack, current command first, following by the environments from "inside" to "outside", meaning from "local" to "global". There is an ID symbol in the left column in addition to the text display.

Double-clicking on one of the lines displayed opens the test module/test step that is referenced in the FTE (a new window opens, if necessary), marks the referenced command there, and moves it into the visible area of the window, if necessary (the block where it is located in collapsed, it will be expanded).
