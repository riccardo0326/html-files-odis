#### <a id="textgruppe.bearbeiten"></a>5.5.1. Manage Text Groups

The editor for creating a text group can be seen in [Figure 75, “Text groups editor”](5-5-1-manage-text-groups.md#figure.textgruppe.neu "Figure 75. Text groups editor"). The user must give a title for the text group.

<a id="figure.textgruppe.neu"></a>

![Text groups editor](images/textgruppen_editor.png)

**Figure 75. Text groups editor**

[Table 21, “Text Group Functions”](5-5-1-manage-text-groups.md#table.textgruppe "Table 21. Text Group Functions") describes the possible functions of the text group.
