#### <a id="uebersetzung.ueberwachen"></a>6.4.8. Monitoring Translations

ODIS Creator displays the translation packages organized by language in the translation navigator. They will be highlighted in color based on the following conditions:

<a id="table.status.%C3%BCbersetzung"></a>

<table border="1" summary="Translation Status"><colgroup><col align="left"/><col/></colgroup><thead><tr bgcolor="#EEEEEE"><th align="left">Condition</th><th>Description</th></tr></thead><tbody valign="middle"><tr><td align="left" valign="middle"><span class="bold"><strong>No highlighting</strong></span></td><td valign="middle">The translation package has not been ordered. The deadline is not impending yet.</td></tr><tr><td align="left" bgcolor="green" valign="middle"> <span class="bold"><strong>Green</strong></span></td><td valign="middle">The translation package must be delivered soon. This gives you an overview of the expected translation packages.</td></tr><tr><td align="left" bgcolor="yellow" valign="middle"> <span class="bold"><strong>Yellow</strong></span></td><td valign="middle">The translation package must be delivered today. This gives you the option to ask the translation provider about time-sensitive translations.</td></tr><tr><td align="left" bgcolor="red" valign="middle"> <span class="bold"><strong>Red</strong></span></td><td valign="middle">The translation package is overdue. The translation package was not delivered on time, and this allows you to take the appropriate actions.</td></tr></tbody></table>

**Table 36. Translation Status**

  

For a better overview, you can limit the translation packages listed in the translation navigator based on their initial deployment deadline through the definition of an individual filter.
