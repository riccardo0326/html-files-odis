### <a id="ocs"></a>5.7. Remote diagnosis export

In ODIS Creator, it is possible to export information about objects so that it can be used within the scope of a cloud application for “remote diagnosis”.

A remote diagnosis export is always paired with a baseline. The export starts automatically after the baseline is created (depending on the configuration of the selected brand). Preparation of automatic increments for complete productive baselines is an exception to this. In that situation, there is not separate remote diagnosis export because its content would not be different from the export of the complete productive baseline.

<a id="figure.ocsexport.overview"></a>

![Image: remote diagnosis export display](images/ocsexport.png)

**Figure 103. Remote Diagnosis Export Display**

The results of the remote diagnosis exports are automatically stored in a central, defined folder.
