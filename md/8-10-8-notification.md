#### <a id="prozess.bereitstellung.pruefung.benachrichtigung"></a>8.10.8. Notification

After an automatically initiated test ends, a notification is created if the generation of notifications for the respective baseline type is activated.

The notification is sent to the respective creator's location administrator team. This message is displayed after the view under the "Order" structure node is refreshed. If the test runs successfully to the end, then this message will read "The plausibility and completeness check 'Name' has ended." If the test is canceled due to errors, then the team will be informed with the following message: "The plausibility and completeness check 'name' was canceled due to errors. Please check the respective log files."

By double clicking on the corresponding notification in the order navigator, you will reach the "deployment" view, where the test navigator opens under the "test" structure node at the respective location and the editor opens for the corresponding test object in read mode.
