### <a id="editoren.zusatzdokumente"></a>7.11. Editors for Supplemental Documents

The supplementary documents can be edited using an internal editor. The internal editor, which can be specified in the settings, is used by default (see [Section 9.1, “Settings”](9-1-settings.md "9.1. Settings")).
