#### <a id="VerbindungMaximaleWartezeitBenutzervorgaben"></a>9.1.1. Settings: Default Value for the Maximum Wait Time

You can configure the default value for the maximum wait time using the settings. The default value is 90000 and can be changed by each user. The value is set for every new command **Send online** or **Receive online**.

<a id="figure.Funktionstesteditor.Arbeitsflaeche.Aktionselemente_Kategorie_Verbindung.Sende_Online.Benutzervorgaben.MaximaleWartezeit"></a>

![Image: default for the maximum wait time](images/fte_Benutzervorgaben_MaximaleWartezeit.png)

**Figure 639. Settings: Default for the Maximum Wait Time**

You can reach the settings via the “Help/Settings” menu item. Select “Function test editor” in the navigation tree to configure the settings for the **maximum wait time**.
