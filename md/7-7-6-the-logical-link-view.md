#### <a id="sgbaum.LogicalLink-View"></a>7.7.6. The Logical Link View

The display of relationships between DTC and control modules is portrayed in an additional view on the log level. The view can be switched using the menu bar in the control module tree. To do this, click on the button for the Logical Link view in the control module tree; see the following image:

<a id="figure.sg-baum.LL-view"></a>

![Image: accessing the Logical Link view](images/LL_View.png)

**Figure 273. Accessing the Logical Link View**

The DTC objects will be compiled under the displayed log levels of the base control modules.

The display of DTC objects in this view is for the creation of suspicion rules and symptom descriptions in FTE. This will make the DTC independent of control module versions.
