#### <a id="auswahl.wissensbasisobjekte"></a>7.19.2. Selection of Knowledge Base Objects

To set the range of the use relationships to be exported, each knowledge base object has a checkbox which can be checked by the author for the P-differ export. This checkbox is included as another property of knowledge bases and diagnostic objects in the properties editor (see [Figure 596, “Selection of Knowledge Base Objects”](7-19-2-selection-of-knowledge-base-objects.md#figure.auswahl.wissensbasisobjekte "Figure 596. Selection of Knowledge Base Objects")). The checkbox can be set by the author, if they have the knowledge base object in processing in the editor. After the author has completed the knowledge base object, the change to the marking is possible through the context menu in the structure tree via the menu functions **Set/remove P-differ export**. During the P-differ export, all referenced test programs are collected resulting from the marked knowledge base objects for the export and for all diagnostic object subnodes available under them.
