#### <a id="Funktionstesteditor.Allgemeine_Funktionalitaet"></a>7.14.2. Menus and Tool Bars

##### <a id="Funktionstesteditor.Allgemeine_Funktionalitaet.Menueleiste"></a>7.14.2.1. Main Menu Bar

The menu bar in the function test editor consists of the menu items for the editing environment and the menu items for the Editor. The functions can be called up using buttons in the menu line of the "Function test editor" screen. The following description of the menu items contains only the categories and actions that are relevant for the function test editor.

The menu bar contains the categories:

- File
- Edit
- View
- Insert
- Extras
- Help

Actions that cannot be performed are grayed out. See also [Figure 335, “Menu Bar”](7-14-2-menus-and-tool-bars.md#figure.Funktionstesteditor.Allgemeine_Funktionalitaet.Menueleiste "Figure 335. Menu Bar").

<a id="figure.Funktionstesteditor.Allgemeine_Funktionalitaet.Menueleiste"></a>

![Image: menu bar](images/fte_Menueleiste.png)

**Figure 335. Menu Bar**

The following actions are under the **File** category:

- Save, see [Save](7-14-3-use-cases.md#Funktionstesteditor.Allgemeine_Funktionalitaet.Speichern "7.14.3.2. Save")
- Combined save, see [Combined Save](7-14-3-use-cases.md#Funktionstesteditor.Allgemeine_Funktionalitaet.KombiniertesSpeichern "7.14.3.3. Combined Save")
