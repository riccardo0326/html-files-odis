### <a id="Funktionstesteditor.Debugger"></a>7.15. Debugger (Integrated Operating System)

The FTE debugger is a sub-component of the function editor. It provides functions for monitored execution of function code objects, for displaying and editing variables in the function code, and for setting break points.

There are various views available when using the debugger.

The debugger component in the function editor allow monitoring of function codes and changes to the runtime. The debugger requires the integrated operating system in order to run.

The menu items **Extras >> "Start test module", "Start test module in Single Step Mode"** and **"Start GFF"** open the debugger with the test module that is currently open in the editor or with the GFF function.
