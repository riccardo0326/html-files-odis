#### <a id="coopexport.setting"></a>8.11.6. Managing a Global Exclusion List

When the “Export settings” node is selected in the navigation tree, the “Export settings” editor [Figure 635, “Collaboration export settings”](8-11-6-managing-a-global-exclusion-list.md#figure.coopexport.settingeditor "Figure 635. Collaboration export settings") will open, where a global exclusion list can be imported or exported.

<a id="figure.coopexport.settingeditor"></a>

![Image: collaboration export settings editor](images/coopexport_setting_editor.png)

**Figure 635. Collaboration export settings**

  

The versioning of the global exclusion list is done internally in OC. In the “Export settings” editor, the date of the last import and the user who performed the import are displayed next to the version. The import logs can also be displayed.
