#### <a id="d4e1311"></a>3.2.8. Separating Views

All views (including navigators, property tabs and history tabs, among others) can be taken out of ODIS Creator and used as a separate window. This layout is not retained after restarting.

If a view is split, then only restricted functions on the objects are possible in this view. Menu and toolbar actions only function in the window they are located in. For example, main menu and main toolbar functions cannot be used in the separated navigator by changing the focus when clicking on the main window. The menu actions located in the view are still available.

If a removed view is added again through **Context menu > Cancel allocation**, then the right view is arranged and not back at the original location. To restore this view, you can select **View > Reset view** or obtain the original layout again using drag and drop.
