#### <a id="Funktionstesteditor.Zusaetzliche_Views.Javacode_View"></a>7.14.24. Java Code (View)<a id="d4e28072"></a>

The codes entered in the test language are displayed in this view. See [Figure 542, “Java Code View”](7-14-24-java-code-view.md#figure.Funktionstesteditor.Zusaetzliche_Views.Javacode_View "Figure 542. Java Code View").

<a id="figure.Funktionstesteditor.Zusaetzliche_Views.Javacode_View"></a>

![Image: function test editor - additional views - Java code view](images/fte_Javacode_View.png)

**Figure 542. Java Code View**

The four buttons in the upper right corner trigger the following functions from left to right:

- Syntax check and compilation
- Compile on server
- Compile for debugging
- Update Java Code
