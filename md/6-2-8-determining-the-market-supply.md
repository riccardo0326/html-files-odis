#### <a id="marktversorung.ermitteln"></a>6.2.8. Determining the Market Supply

The order management offers you, as the author or editing coordinator, the option to check if all translations are available for your projects. To do this, use the multiple selection function (Ctrl or Shift button) in the order management navigator to select order headers and/or order steps that contain editorial objects required for the review. Then select the find function in the global menu under **Edit >> Find missing translation** or **Find missing translation** in the context menu.

The computed order steps from your selection will be listed in the dialog from [Figure 131, “Dialog for Determining the Market Supply”](6-2-8-determining-the-market-supply.md#figure.marktversorgung.ermitteln.dialog "Figure 131. Dialog for Determining the Market Supply"). Here you must still specify the target date for the translations. You will start the inquiry process in the background by selecting the "OK" button. You can stop the inquiry using the "Cancel" button.

<a id="figure.marktversorgung.ermitteln.dialog"></a>

![Image: dialog for finding the market supply](images/marktversorgung_dialog.png)

**Figure 131. Dialog for Determining the Market Supply**

The found result will be entered in an Excel file. This depends on the notification, that is received through the standard mechanism (see [Section 6.1.3, “Notifications”](6-1-3-notifications.md "6.1.3. Notifications")). You will see the example structure of the Excel file in [Figure 132, “Example Excel File for Determining the Market Supply”](6-2-8-determining-the-market-supply.md#figure.marktversorgung.excel "Figure 132. Example Excel File for Determining the Market Supply").
