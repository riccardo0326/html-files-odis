#### <a id="xml.templates.navigator"></a>7.12.2. Navigator for XML Templates

The XML templates will be displayed in a separate navigator corresponding to the other editorial objects (see [Figure 325, “Navigator for XML Templates”](7-12-2-navigator-for-xml-templates.md#figure.xml.template.navigator "Figure 325. Navigator for XML Templates")). Because there is no superordinate organizational order, such as for the supplemental documents, then all XML templates will be displayed without root nodes. The display is sequenced according to how the XML templates were created. If you have selected another XML template before creating a new XML template, the new object will be inserted directly after this XML template. This process is identical to the other navigators in ODIS Creator. The rest of the functions also match those in the other navigators. You can place the XML template in processing by double clicking or through menu selection. Otherwise the same functions are available to you through the context menu, as with all other navigators. You can also delete XML templates and initiate the standard release process for editorial objects. The detail display of XML templates in the editor and the allocation of XML templates to function codes is possible for all users, who have the author role. For all other functions, the location administrator role must also be assigned to you. The visibility of the displayed XML template is handled the same as in the navigator for supplemental documents. All XML templates will be displayed that either have a release status or are being edited by your own team.
