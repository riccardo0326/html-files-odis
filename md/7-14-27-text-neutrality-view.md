#### <a id="Funktionstesteditor.Zusaetzliche_Views.Textneutralitaet_View"></a>7.14.27. Text Neutrality (View)

This view displays the result of the text neutrality check in the order of usage. This view consists of two columns: Icon column for the icon of the action element and Result column for the name of the action element.

<a id="figure.redaktion.textneutralit%C3%A4t.reiter"></a>

![Figure: Text neutrality tab](images/reiter_textneutralitaet.png)

**Figure 549. Text Neutrality Tab**

If translation-relevant texts are available, the affected point is listed in the text neutrality view and highlighted in the test sequence and can be accessed via the entries in the text neutrality tab. The arrow keys also enable navigation through the entries in the table.

When the function code is saved and checked in, the validation check is executed again and the final status is saved in the database.

##### <a id="Funktionstesteditor.Zusaetzliche_Views.Textneutralitaet_View_aktualisieren"></a>7.14.27.1. Update text neutrality check

This function is used to list the check in the text neutrality view.

##### <a id="Funktionstesteditor.Zusaetzliche_Views.Textneutralitaet_View.filter"></a>7.14.27.2. Filtering Text-Neutral Objects

The function library navigator offers the option to filter by “text neutrality” to expand or restrict the number of objects displayed in the navigator. The property “text neutrality” has been added to the filter function in the editor. The possible filter conditions are ‘Equal’ (=) and 'Unequal' (<>) and the filter options are ‘Yes’, ‘No’, or ‘Not calculated’. A detailed description of the [filter definitions](3-2-3-navigator.md#navigator.filter.definieren "3.2.3.1. Defining the filter") and [filter conditions](3-2-3-navigator.md#table.filterbedingungen "Table 10. Filter Conditions") is displayed under [Navigators](3-2-3-navigator.md "3.2.3. Navigator").
