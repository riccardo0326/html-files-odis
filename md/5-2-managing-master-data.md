### <a id="stammdaten.verwalten"></a>5.2. Managing Master Data

The master data is managed in the **Master data navigator** view in a tree structure ([2](5-2-managing-master-data.md#figure.stammdatennavigator "Figure 47. Master Data Navigator")).

Clicking on an entry opens a linked table or text field with contents in the editor ([1](5-2-managing-master-data.md#figure.stammdatennavigator "Figure 47. Master Data Navigator")). If you would like to edit the contents of a [table](2-2-general-definitions-and-glossaries.md#tabelle), the editor must be opened in [editor mode](3-2-5-editor.md "3.2.5. Editor").

The **master data properties** ([3](5-2-managing-master-data.md#figure.stammdatennavigator "Figure 47. Master Data Navigator")) are displayed in the [Properties window](3-2-4-properties-window.md "3.2.4. Properties window").
