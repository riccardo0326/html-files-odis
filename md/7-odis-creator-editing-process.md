## <a id="prozess.redaktion"></a>7. ODIS Creator Editing Process

The **Editing** module is the central component of the ODIS Creator editing system. It serves not only for the creation and management of editorial objects for the diagnosis, but rather also ensures their consistency, and provides the translation with translation candidates<a id="d4e5658"></a>, has interfaces for a standard test language and the editing systems in context (for example, LIVAS 3, MOVE, etc.). It integrates internal and external content editors (such as the function test editor, Epic as XML editor, hotspot editor, etc.) and has interfaces for order management and administration as well as for supplying.

The editing module is assigned to the author role. The following views are available to the author:
