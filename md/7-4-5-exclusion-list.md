#### <a id="ausstattungsnetz.ausschlussliste"></a>7.4.5. Exclusion List

Importer numbers (e.g. “128” or “Global”) can be maintained in the Exclusion list table (see [Figure 237, “Exclusion List Table”](7-4-5-exclusion-list.md#figure.redaktion.ausstattungsnetz.ausschlussliste "Figure 237. Exclusion List Table")) so that this information can be used later for the exclusion lists for deployments.

The individual rows allow a maximum of 10 characters. If more than 10 characters are entered, the user is informed of this by means of a dialog when accepting; only the first 10 characters are applied. Blank entries are not allowed and will be removed automatically during saving.

In the table, it is possible to change the sorting and filtering via the column header. Filtering is performed via a context menu.

The table is available in the following equipment network objects:

- Vehicle type
- Model year
- Version
- Motor

##### <a id="ausstattungsnetz.ausschlussliste.filter"></a>7.4.5.1. Filtering Importers

The equipment network navigator offers the option to filter by “importer” to expand or restrict the number of vehicle types displayed in the navigator. The free text search is not case-sensitive. A detailed description of the [filter definitions](3-2-3-navigator.md#navigator.filter.definieren "3.2.3.1. Defining the filter") and [filter conditions](3-2-3-navigator.md#table.filterbedingungen "Table 10. Filter Conditions") is displayed under [Navigators](3-2-3-navigator.md "3.2.3. Navigator").
