#### <a id="redaktion.marken"></a>7.1.10. Editorial Object Brand Codes

All editorial objects, with the exception of control module descriptions, are assigned to one or more brands in ODIS Creator. This occurs as soon as an editorial object is directly or indirectly assigned to an equipment characteristic. This brand allocation influences the ability to edit, use and view an editorial object.

The brand codes for object brands can be viewed in the navigation trees/libraries, in the selection dialogs and also in the respective object editors. This display can be viewed for the following object types:

- Knowledge base object types
- Function library object types
- Supplemental documents
- XML Templates
- Control Module Configuration
- Equipment without diagnostic interface
- XOR Nodes

The display of brand codes is given as a list of respective brand codes (KVPS names) separated by commas (see [Figure 188, “Brand Code in Object Editor”](7-1-10-editorial-object-brand-codes.md#figure.redaktion.markenkuerzel "Figure 188. Brand Code in Object Editor")). This list is sorted alphabetically.

<a id="figure.redaktion.markenkuerzel"></a>

![Image: brand code in object editor](images/Markenkuerzel_redaktion.png)

**Figure 188. Brand Code in Object Editor**

In addition to the editorial object editors, the brand allocation information is also displayed in the navigation trees (see [Figure 189, “Brand Code in Navigator”](7-1-10-editorial-object-brand-codes.md#figure.redaktion.markenkuerzel.tooltip "Figure 189. Brand Code in Navigator")), libraries and all selection dialogs (see [Figure 190, “Brand Code in Selection Dialog”](7-1-10-editorial-object-brand-codes.md#figure.redaktion.markenkuerzel.tooltip.auswahl "Figure 190. Brand Code in Selection Dialog")) for all editorial object types mentioned. The brand allocation is also displayed in a tool tip. If the user moves the mouse cursor over an object displayed in the navigator or selection dialog, then a tool tip will open with the brand allocation displayed as a list of brand codes separated by commas.
