#### <a id="xml.templates.fte.perspektive"></a>7.12.4. Usage in the Function Library View

The selection of XML templates is done according to the selection of editorial objects within the function library view. There is a text field with read rights for this, with a button on the right side. The selection dialog for editorial objects will open when the button is pressed, in which all XML templates that the author is allowed to see will display. These are all released XML templates. The author will also see working versions of XML templates, that are assigned to their team.
