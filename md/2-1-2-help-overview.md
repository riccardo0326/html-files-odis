#### <a id="hilfe.uebersicht"></a>2.1.2. Help Overview

So that you can find your way easily in ODIS Creator Help, the interface follows the popular design of Windows Help. You will find the [selection menu](2-2-general-definitions-and-glossaries.md#auswahlmenu) with the ODIS Creator symbol on the left side ([1](2-1-2-help-overview.md#figure.hilfe.uebersicht "Figure 2. Help Overview")) and the symbols to minimize, maximize or close the [window](2-2-general-definitions-and-glossaries.md#fenster) are on the right side ([2](2-1-2-help-overview.md#figure.hilfe.uebersicht "Figure 2. Help Overview")). There is a bar under that where the search function can be found ([3](2-1-2-help-overview.md#figure.hilfe.uebersicht "Figure 2. Help Overview")). In the left [frame](2-2-general-definitions-and-glossaries.md#rahmen) ([4](2-1-2-help-overview.md#figure.hilfe.uebersicht "Figure 2. Help Overview")), you can open four different views through the lower register tab ([5](2-1-2-help-overview.md#figure.hilfe.uebersicht "Figure 2. Help Overview")) (**Table of Contents**, **Search results**, **Links to related topics**, **Bookmark**). In the right frame, the selected help texts will be displayed ([6](2-1-2-help-overview.md#figure.hilfe.uebersicht "Figure 2. Help Overview")).

##### <a id="eclipse.hilfe"></a>2.1.2.1. Using the Eclipse Help Function

In the frame on the left side, the **Content** tab (![symbol: content](images/contents_view.gif)) contains the **content folder**. Click on a topic to display it in the right frame ([6](2-1-2-help-overview.md#figure.hilfe.uebersicht "Figure 2. Help Overview")).

With the arrow keys ![](images/Pfeiltaste_zurueck.png) and ![](images/Pfeiltaste_vor.png) usage steps within help can be reversed or restored incrementally.

<a id="eclipse.hilfe.suchen"></a>**Search**
:   To search for a specific topic, you can enter a search term in the **Search** field. You can use the **search area** button to limit the search to topic areas that you are interested in. The **GO** button starts the search. In the left frame, the **Search results** tab (![symbol: search](images/search_results_view.gif)) will automatically open. Clicking on a search result opens the topic in the right frame.

<a id="eclipse.hilfe.synchronisieren"></a>**Synchronize**
:   If a topic was found and searched sufficiently, you can click on the button **Update / Display current topic** (![symbol: Update / Display current topic](images/e_synch_nav.gif)) or click on the button **Display in content folder** (![symbol: Display in content folder](images/e_synch_toc_nav.gif)) to open the **Content** tab again in the left frame. The currently opened topic is displayed in the content folder structure tree.

<a id="eclipse.hilfe.funktionalitaet"></a>**Functionalities**
:   If you want to display the documentation for functionalities, that are disabled in the application, select the button **Display all topics** (![symbol: Display all topics](images/e_show_all.gif)). If you select that all topics should be displayed in the content folder, then the headings of the documentation will also be displayed in disabled activities in the content folder and in the search results.

##### <a id="vaudes.hilfe"></a>2.1.2.2. Navigation within an Open Section

To navigate in an open section, additional [buttons](2-2-general-definitions-and-glossaries.md#schaltflaeche) are given in ODIS Creator Help. The buttons are in both the header and footer sections. **Next** ([1](2-1-2-help-overview.md#figure.hilfe.uebersicht2 "Figure 3. ODIS Creator Help Navigation")) and **Back** ([2](2-1-2-help-overview.md#figure.hilfe.uebersicht2 "Figure 3. ODIS Creator Help Navigation")) switch to the next or previous section. The button **Back to top** ([3](2-1-2-help-overview.md#figure.hilfe.uebersicht2 "Figure 3. ODIS Creator Help Navigation")) switches to the beginning of a section and the button **To start** ([4](2-1-2-help-overview.md#figure.hilfe.uebersicht2 "Figure 3. ODIS Creator Help Navigation")) opens the section overview for help.

Colored and underlined texts ([5](2-1-2-help-overview.md#figure.hilfe.uebersicht2 "Figure 3. ODIS Creator Help Navigation")) are links to sections with additional information. Clicking on this type of text will take you directly to the section with further information about the linked text. Clicking on the upper left back arrow ![](images/uebersicht_hilfe2_zurueck.png) ([6](2-1-2-help-overview.md#figure.hilfe.uebersicht2 "Figure 3. ODIS Creator Help Navigation")) switches back to the linked text passage.

<a id="figure.hilfe.uebersicht2"></a>

![Image: ODIS Creator Help navigation](images/uebersicht_hilfe2.png)

**Figure 3. ODIS Creator Help Navigation**

To describe functions clearly, additional image references are contained in the text. Clicking on an image reference within a help text takes you directly to the selected image ([Figure 4, “Target Link of an Image Reference”](2-1-2-help-overview.md#figure.beispiel "Figure 4. Target Link of an Image Reference")).
