### <a id="hilfe.bedienungsanleitung"></a>2.1. User Guide

The ODIS Creator application is based on the platform [Eclipse](11-appendix.md#eclipse "Eclipse")<a id="d4e101"></a>which is a development system for software. The ODIS Creator Help is integrated as a topic in the Eclipse Help. In addition to the fundamental functions of Eclipse Help, ODIS Creator Help offers additional options for navigation.
