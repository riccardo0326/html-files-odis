#### <a id="baseline.produktiv.browser"></a>8.1.1. Production Baseline Browser

The production baseline browser shows all available production baselines in a tree structure (see [Figure 601, “Production Baseline Browser”](8-1-1-production-baseline-browser.md#figure.produktiv.baseline.browser "Figure 601. Production Baseline Browser")). You have the option to display the production baseline properties (one mouse click) and to edit them (two mouse clicks).

The navigator tree of the production baseline browser is hierarchically structured according to brands and year of creation.
