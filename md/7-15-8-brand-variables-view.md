#### <a id="Funktionstesteditor.Debugger.MarkenVariablen_View"></a>7.15.8. Brand Variables View

The brand variables view shows the used brand variables in the debug session and their values in a tabular format.

<a id="figure.Funktionstesteditor.Debugger.MarkenVariablen_View"></a>

![Image: brand variables view](images/fte_Debug_MarkenvariablenView.png)

**Figure 580. Brand Variables View**

The values of brand variables cannot be changed in the brand variables view. However, the function test variables assigned to a brand variable can be changed through the variables view. See [Section 7.15.7, “Variables View”](7-15-7-variables-view.md "7.15.7. Variables View").

The brand variable values are updated at the following times:

- Test program start
- Test program end
- Sub-program call up (including return)
