#### <a id="grundlagen.datumsfelder"></a>3.2.9. Date Fields

Date inputs are displayed at many locations in the ODIS Creator system. The time will also be displayed for objects that also have a concrete time stored to it. The format that includes the time is **"dd.MM.yyyy, HH:mm:ss"**, which can be seen in the example [Figure 44, “Properties Window for Editorial Objects”](3-2-9-date-fields.md#figure.eigenschaftenview "Figure 44. Properties Window for Editorial Objects") or [Figure 32, “Properties Window for Orders”](3-2-4-properties-window.md#figure.sortnone "Figure 32. Properties Window for Orders"). At all other locations where a concrete time is not available, the format is **"dd.MM.yyyy"**.

<a id="figure.eigenschaftenview"></a>

![Image: properties window for editorial objects](images/eigenschaftenview.png)

**Figure 44. Properties Window for Editorial Objects**

The date entries in the selection fields, for example in the filter dialog, are only ever date entries based on days. This means that no explicit time sample can be selected there.
