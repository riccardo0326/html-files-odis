#### <a id="markenvariable.filtern"></a>7.13.8. Brand Variable Filters

The filter in the "brand variables" navigator functions the same as in other locations, such as the [editor](7-1-5-apply-filter.md "7.1.5. Apply Filter"). Brand variables are filtered under the brand nodes.

The brand variable attributes that can be used for the filter can be found in [Table 67, “Brand Variable Filter Attributes”](7-13-8-brand-variable-filters.md#table.markenvariable.filtern "Table 67. Brand Variable Filter Attributes").
