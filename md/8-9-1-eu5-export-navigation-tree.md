#### <a id="eu5export.navigator"></a>8.9.1. EU5 Export Navigation Tree

The navigation tree for EU5 exports shows all available EU5 exports in a tree structure. You have the option to display the properties of EU5 exports (one mouse click).

The EU5 exports are hierarchically structured in the tree based on the year they were created.

The status of EU5 exports are taken from the status model for production baselines. The status [**Created**](8-6-1-created.md "8.6.1. Created"), [**In processing**](8-6-2-in-progress.md "8.6.2. In progress"), [**Completed**](8-6-8-completed-externally.md "8.6.8. Completed (Externally)") and [**Error**](8-6-4-error.md "8.6.4. Error") are permitted for EU5 exports.

New EU5 exports can be created through the context menu **Export definition**. The editor opens in write mode by clicking on it, and then the required entries can be made.

All EU5 exports can be deleted from the database through the context menu **Delete**. This will also deleted the export results. Deleting EU5 exports with the [**In processing**](8-6-2-in-progress.md "8.6.2. In progress") status is not possible.
