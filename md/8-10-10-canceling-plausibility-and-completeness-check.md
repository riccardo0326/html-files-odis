#### <a id="prozess.bereitstellung.pruefung.abbrechen"></a>8.10.10. Canceling Plausibility and Completeness Check

The location administrator can select multiple test objects in the test tree. A menu item "Cancel test" is given in a test object context menu, which can be used to initiate the cancellation of a test. A confirmation dialog will be displayed before canceling the previously selected test objects, informing that the tests will be canceled and cannot be restarted. Only test objects with the status "created" or "running" can be canceled. Since the test runs as a background process, the cancellation of a test will not be performed immediately but rather with a delay.
