# **Elenco acronimi – ODIS Creator**

# **A**

**ALM**

*Application Lifecycle Management*

Strumento (HP ALM) usato per la gestione di Requirements, Test Coverage e Status.

**ASAM**

*Association for Standardisation of Automation and Measuring Systems*

Organizzazione che definisce standard per la diagnostica automotive.

**ASAM ECUKOM / EKUCOM**

Modulo ODIS per la comunicazione diagnostica standardizzata con le ECU.

# **B**

**BV**

*Basic Variant*

Struttura diagnostica di base del progetto veicolo; aggrega EV, GFF e gruppi di controllo.

# **C**

**CAN**

*Controller Area Network*

Bus di comunicazione veicolare.

**CTRL**

*Tasto Control*

Usato per multiselezione di elementi non adiacenti in ODIS Creator.

# **D**

**DA**

*Diagnostic Address*

Indirizzo diagnostico che identifica una ECU; oggetto diagnostico reale.

**DEV**

*Development*

Ambiente di sviluppo (non LIVE).

**DID**

*Data Identifier*

Identificatore UDS per dati diagnostici letti/scritti dalla ECU.

**DTC**

*Diagnostic Trouble Code*

Codice di guasto diagnostico.

# **E**

**ECU**

*Electronic Control Unit*

Centralina elettronica del veicolo.

**EV**

*Electronic Variant*

Variante diagnostica specifica della centralina; contiene DTC e associazioni GFF.

# **F**

**GF**

*Guided Function*

Funzione diagnostica guidata generica.

**GFF**

*Guided Fault Finding*

Funzione diagnostica reattiva ai DTC.

# **H**

**HP ALM**

*Hewlett-Packard Application Lifecycle Management*

Sistema di gestione Requirements e Test Coverage.

# **I**

**IF**

Costrutto logico condizionale usato nelle test sequence.

# **K**

**KBN / KB**

*Knowledge Base*

Struttura che contiene oggetti diagnostici e funzioni.

# **L**

**LIVE**

Ambiente di produzione/rilascio ufficiale.

# **O**

**OC**

*ODIS Creator*

Strumento di authoring diagnostico.

**ODIS**

*Offboard Diagnostic Information System*

Piattaforma diagnostica del Gruppo Volkswagen.

**ODX**

*Open Diagnostic Data Exchange*

Formato standard ASAM per dati diagnostici.

**OM**

*Order Management*

Modulo per gestione release, hotfix, complete e risk release.

# **P**

**PDX**

*Package Data Exchange*

Pacchetto diagnostico che contiene EV/ODX rilasciati.

# **R**

**REQ**

---

*Requirement*

Requisito tracciato in HP ALM.

# **S**

**SFD**

*Schutz Fahrzeug Diagnose* (Vehicle Diagnostic Protection)

Sistema di protezione diagnostica con lock/unlock.

**SPG**

*Signal Processing Group*

Prefisso tipico di oggetti diagnostici legati a condizioni o segnali.

**SIG**

*Signal*

Oggetto diagnostico relativo a un segnale.

**SHIFT**

*Tasto Shift*

Usato per multiselezione di elementi adiacenti.

# **T**

**TBL**

*Test Baseline*

Baseline di test che aggrega tutte le modifiche diagnostiche.

# **U**

**UDS**

*Unified Diagnostic Services*

Protocollo diagnostico standard ISO 14229.

**Usage / Usage Calculation**

Calcolo che collega GFF, DTC, EV e BV per determinare copertura.

# **V**

**VLOOKUP**

Funzione Excel usata per confrontare elenchi DTC e verificare copertura.

# **Z**

**ZDC**

*Zonal Data Container*

Contenitore dati target in System42 associato a una ECU.
