### <a id="didb.directory"></a>9.5. DIDB

The current [**DIDB**](11-appendix.md#didb "DIDB") to be used is saved in the sub-folder "DIDB" of the ODIS Creator installation folder (normally can be found under "C:\webApps\ODISC"). For additional information about using DIDB, please refer to the ODIS Service documentation.

**Note:** When starting ODIS Creator through the Web Starter, the folder "C:\webApps\ODISC\DIDB" is deleted!
