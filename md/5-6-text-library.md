### <a id="textbib"></a>5.6. Text Library

The default texts in the text library are created and managed in ODIS Creator. Text IDs or text modules are sorted into categories. The standardized text or a placeholder for a variable or keyword may be in a text ID. A text module is a sequence of various text IDs from different categories.

<a id="figure.textbib.navigator"></a>

![Text library view](images/textbib_perspektiven.jpg)

**Figure 80. Text library view**

The management of categories and text IDs or text modules is assumed by the [text administrator](3-3-1-description-of-roles.md "3.3.1. Description of Roles") role. The views "text library" and "text library settings" are in the administration (see [Figure 80, “Text library view”](5-6-text-library.md#figure.textbib.navigator "Figure 80. Text library view")).

The category settings, such as text formats or usage, are managed in the "text library settings" navigator. Both editors "text formats overview" and "usage locations overview" are also given here. Both of these editors give an overview of every category and their settings.

Categories and text IDs are managed in the "text library" navigator.
