#### <a id="prozess.bereitstellung.pruefung.loeschen"></a>8.10.9. Deleting Plausibility and Completeness Check

The location administrator can select multiple test objects in the test tree. A menu item "Delete test" is given in a test object context menu, which can be used to initiate the deletion of a test. A confirmation dialog will be displayed before deleting the previously selected test objects, informing that the selected tests will be deleted and cannot be restored. Only test objects with the status "completed", "canceled" or "error" can be deleted. The log saved in the database and the test object itself are deleted. For the manual tests, the log is also removed from the archive folder.
