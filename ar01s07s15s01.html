7.15.1. License for Debugger / ODIS Service

The integrated ODIS Service Debug operating system enables you to import a license file. The operating system uses the license to ensure access of ODIS Debug to group systems, for example. A message about missing license information will appear when the debug session starts.

Figure 550. License Missing


The import is done using the ODIS Debug configuration in the global menu under Extras >> ODIS Debug >> ODIS Debug configuration. A license for ODIS Debug can be extracted from an ODIS Service license using the Extract license information button. The license is stored in the path C:\webapps\odisc\License.xml and used in subsequent ODIS Creator sessions on the user PC.
