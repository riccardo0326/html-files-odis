#### <a id="redaktion.unterknoten.sortieren"></a>7.1.6. Sorting Subnodes

In addition to sorting through all object in a navigator, you can also sort only the sub-objects for an editorial object. This is persistent in contrast to sorting in the navigator, therefore the order of sub-objects is saved in the database.

In the menu, select **Edit > Sort subnodes** to do this. A dialog appears if you would like to sort subnodes alphabetically.
