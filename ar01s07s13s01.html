7.13.1. Creating New Brand Variables

You can create a new brand variable, by selecting the brand node or a brand variable and then clicking on the menu item Create brand variable or the Create brand variable in the insert selection menu. It will open in the editor, shown in Figure 330, “Brand variable editor”.

Figure 330. Brand variable editor


A brand variable can be a single variable or an array. As soon as the option field Array is selected, an array length must also be given. In Figure 331, “Brand Variable Field Size”, you will see an array variable with a specified length.

Figure 331. Brand Variable Field Size
