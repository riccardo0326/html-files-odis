6.3.2. Text Request Filters

In the “Text request” navigator, text requests can be filtered in order to expand or restrict the number of text requests displayed in the navigator.

In the illustration that follows, the button for filtering text requests and the filter dialog are highlighted. After entering the filter criteria in the filter dialog, these criteria can be used directly or saved as a filter that can be selected later. Clicking the OK button in the filter dialog applies the filter. Clicking the filter button again deactivates the filter so that text requests can be displayed unfiltered in the navigator.

Figure 144. Text Requests Filter


Entries can be filtered by the following criteria:

  • Last editor

  • Editing date

  • Creation date

  • Status

  • Status date

  • Type (new entry or edited entry)
