#### <a id="tabellen.listen"></a>3.2.6. Tables and Lists

In some tables,<a id="d4e1202"></a>the order of the table rows can be changed ([Figure 37, “Table Sorting”](3-2-6-tables-and-lists.md#figure.tablesort "Figure 37. Table Sorting")). To do this, click on the column header of the column containing the entries which the table should be resorted by. Based on the entries in the selected column, the table lines are displayed in ascending, descending or unsorted alphanumeric order.

<a id="figure.tablesort"></a>

![Image: table sorting](images/TableSort.png)

**Figure 37. Table Sorting**

In some tables or lists you can create new lines, in which you click on the plus symbol on the right side above a table or list. If you would like to change an already existing entry, click directly in the respective line. To delete a line, mark it and then click on the minus symbol.

In the [editing](7-odis-creator-editing-process.md "7. ODIS Creator Editing Process") process, tables serve only to display contents. The cells cannot be edited directly. New data sets are created using the [button](2-2-general-definitions-and-glossaries.md#schaltflaeche) ![Add](images/plus.png) . Then you have to option to create a link to the Internet, in the [context menu](2-2-general-definitions-and-glossaries.md#kontextmenu) for example. A supplemental editor is loaded, in which the contents are entered and then applied.

<a id="d4e1224"></a>

![Image: Internet editor](images/InternetEditor.png)

**Figure 38. Internet Editor**
