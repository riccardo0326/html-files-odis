#### <a id="baseline.state.extern.inProgress"></a>8.6.7. In External Processing

This status indicates a baseline in external processing, for which the status is given as "Created" or "In processing" in the existing status file.

The icon for this status shows ![](images/baseline_in_progress_external.png) a gear with a clock.
