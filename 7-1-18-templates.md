#### <a id="redaktion.vorlagen"></a>7.1.18. Templates

The template navigator can be accessed in the editing view through the menu **Extras >> Templates**. The template objects are organized in the template navigator by types (see [Figure 214, “Template Navigator”](7-1-18-templates.md#figure.redaktion.vorlagen "Figure 214. Template Navigator")).

<a id="figure.redaktion.vorlagen.menu"></a>

![Image: templates menu item](images/vorlagen_menu.png)

**Figure 213. Templates Menu Item**

For objects from the equipment network, function library, knowledge base and supplemental documents, a template can be created and then used to create an editorial objects.

<a id="figure.redaktion.vorlagen"></a>

![Image: template navigator](images/vorlagen.png)

**Figure 214. Template Navigator**

##### <a id="d4e7124"></a>7.1.18.1. Creating a Template

There are two ways to create a template.

- In the context menu of an editorial object, you will find the option **Use as template** (see [Figure 215, “Use as Template”](7-1-18-templates.md#figure.redaktion.vorlagen.setzen "Figure 215. Use as Template")) if it is possible for the object.
- A template can be created directly in the template navigator. To do this, open the context menu of the desired object type and select **New template >> desired editorial object**. It will open the editor for the selected object.

<a id="figure.redaktion.vorlagen.setzen"></a>

![Image: use as template](images/vorlage_setzen.png)

**Figure 215. Use as Template**

##### <a id="d4e7143"></a>7.1.18.2. Using a Template

To use an already prepared template to create a new editorial object, pull the template to the respective navigator at the desired location using drag and drop. For example, a knowledge base template in the knowledge base navigator between two knowledge bases, or a diagnostic object to the desired knowledge base.
