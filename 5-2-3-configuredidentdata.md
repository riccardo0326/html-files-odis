#### <a id="administration.configuredidentdata"></a>5.2.3. ConfiguredIdentData

The ConfiguredIdentData.properties file is edited in the ConfigureIdentData editor so that it can be exported into a separate “ConfiguredIdentData” folder below the global folder during preparations (production [complete and incremental], rollout, corrections, except for test baseline and no hotfix).

The checksum for the ConfiguredIDentData.properties file is also calculated at each save. This checksum is displayed in the editor and the properties window and is stored in a separate ConfiguredIdentData.properties\_checksum.txt file in the same folder as the ConfiguredIdentData.properties file for the relevant deployment types.
