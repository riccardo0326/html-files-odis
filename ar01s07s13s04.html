7.13.4. Edit brand variable

Depending on the brand variable status as well as if the brand variable is already used, only limited editing options may be permitted. As soon as the brand variable is used at least one time, the system name, variable type, log and variable/array are not changed anymore. If the brand variable is defined as array, then the array length only increases. If all usage locations are deleted, a change is possible again as long as the brand variable is in the "In processing" status. Changes to brand variables affect its usage locations first, if the declare dialog is edited within the respective function code objects.
