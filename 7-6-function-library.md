### <a id="funktionsbibliothek"></a>7.6. Function Library<a id="d4e13379"></a>

The function library refers to all functional objects. It allows direct access to it and its list. The functional objects are arranged in the [navigation tree](7-6-function-library.md#figure.redaktion.funktionsbibliothek.navigation "Figure 267. Function Library Navigation Tree") based on their type. The types of functional objects may be separated in two categories; one group of the objects can only be created in the function library ([measured value tables](7-6-2-creating-and-changing-measured-value-tables.md "7.6.2. Creating and Changing Measured Value Tables"), [test procedures](7-6-1-creating-and-changing-test-procedures.md "7.6.1. Creating and Changing Test Procedures"), [preset measurements](7-6-3-creating-and-changing-default-measurements.md "7.6.3. Creating and Changing Default Measurements")), and the other group of objects can also be created in other navigation trees in editing ([function test](7-6-4-function-code-objects.md#funktionscode.funktionstest "7.6.4.3. Creating and Changing a Function Test"), [general test](7-6-4-function-code-objects.md#funktionscode.generellertest "7.6.4.5. Creating and Changing General Tests"), [traversal test](7-6-4-function-code-objects.md#funktionscode.traversierungstest "7.6.4.4. Creating and Changing a Traversal Test")). Objects in the second group can be created there, where they are technically integrated. Further information can be found in the following sections. It does not matter in which navigation tree context a functional object is created, the functional library shows you the entirety of all objects of the corresponding type visible to you.

<a id="figure.redaktion.funktionsbibliothek.navigation"></a>

![Image: function library navigation tree](images/funktionsbiliothek_navigationsbaum.png)

**Figure 267. Function Library Navigation Tree**

  

The separate access to the functional objects permits the allocation of special access rights to this section.

The function library is the data section for the function test editor.
